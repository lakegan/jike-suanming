# Jike Suanming

A project dedicated to providing insightful calculations through Bazi and AI integrations.

## Getting Started

1. Clone the repository:
   ```bash
   git clone https://github.com/your_username/jike-suanming.git
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
